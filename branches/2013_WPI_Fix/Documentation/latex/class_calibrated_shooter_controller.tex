\subsection{\-Detailed \-Description}
\-Allows you to adjust and test the precise speeds of the shooter via \-Smart\-Dashboard. 

\-This lets you precisely calibrate the speed of the motors via the \-Smart\-Dashboard, then shoot it by using a joystick.
