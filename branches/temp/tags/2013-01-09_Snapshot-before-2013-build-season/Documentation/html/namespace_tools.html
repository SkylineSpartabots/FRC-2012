Takes a number between a certain range and scales it within another range.

For example, if I had 

 Tools::Coerce(3, 0, 10, 0, 100);

,

The code would see that '3' is %30 percent the range 0 to 10, and then return 30 percent of the range from 0 to 100.

Examples: 

 Tools::Coerce(3, 0, 10, 0, 100);               // 30
 Tools::Coerce(4, 0, 10, 10, 110);      // 40
 Tools::Coerce(4, 1, 11, 200, 400);     // 260

Truncates a number if it exceeds a range.

For example, if I had: 

 Tools::Limit(3.0, 1.0, 2.0);

...the function would return 2.0 because 3.0 exceeds 2.0.

However, if I had: 

 Tools::Limit(0, -1.0, 100.5);

...the function would return 0 because it lies between -1.0 and 100.5.

Parameters:
[in]numberThe number to check
[in]lowestThe lowest number allowed
[in]highestThe highest number allowed
Returns:
Returns the truncated number.
