\subsection{\-Detailed \-Description}
\-A base class meant to be inherited from any component class. 

\-Any class that controls a specific piece of the robot should be a subclass of this.

\-Inheriting this class adds no extra functionality for now -\/-\/ it's simply a method to help organize the code.

\-Additional functionality may be provided in the future.
