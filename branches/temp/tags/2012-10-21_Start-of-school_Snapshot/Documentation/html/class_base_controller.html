Detailed Description

A base class meant to be inherited from any controller class.

Any class that controls components by using user input should be a subclass of this.

All subclasses need to provide a 'Run' method -- the Run method cannot contain any excessively long loops or waiting periods. During the normal operation of a robot, multiple Run methods from different classes will be repeatedly called in a loop until the robot is disabled.
