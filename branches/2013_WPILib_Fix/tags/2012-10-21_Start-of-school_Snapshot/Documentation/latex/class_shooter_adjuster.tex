\subsection{\-Detailed \-Description}
\-Adjusts the robot to face the target more accurately. 

\-This will take command over the drive system temporarily, and will disable or feed the watchdog during the duration.

\begin{DoxyWarning}{\-Warning}
\-Using this class will pause the robot for approximately 0.\-5 seconds to obtain the image, then a few more seconds to move the robot into the correct position. 
\end{DoxyWarning}
