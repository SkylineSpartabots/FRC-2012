Detailed Description

Takes a single joystick and drives the robot, arcade style.

Warning:
Use TankJoysticks() -- this class is slowly starting to become depreciated.

Joystick *  joystick 
)

A method of controlling the robot by using only one joystick.

This is a form of controlling the robot by using the single Extreme 3D Pro joystick. This differs from our other joysticks because it comes with twisting motions.

Parameters:
[in]robotDriveA pointer to the robotdrive
[in]joystickA pointer to the joystick (use the Extreme 3d Pro joystick)
